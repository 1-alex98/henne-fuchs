How to play

How chickens move

  • Chickens move one cell to an empty connected cell.
  • Chickens can move left, right, and up (towards the stall area).
  • Chickens cannot move down.
  • There is no jumping/capturing move for chickens in this implementation.

How foxes move

  • Foxes move one cell to an empty connected cell.
  • Foxes can move left, right, up, and down.
  • On some positions, diagonal connections exist, so a fox may also move diagonally when that connection is present.

How foxes jump (capture)

  • If a fox has an adjacent chicken, and the cell immediately beyond that chicken (in the same direction) is empty, the fox can jump there.
  • The jumped chicken is removed from the board.
  • Foxes can chain jumps; the best jump is the one that continues until no further jump is possible.
  • If any fox can jump, foxes are expected to jump. If a fox moves instead, it is punished.
  • If a fox starts a jump but does not continue to the end of a possible chain, it is punished.

How chickens win

  • Chickens win when 9 or more chickens are in the stall area.

How foxes win

  • Foxes win when the number of chickens left is less than 9 (enough chickens have been removed).

How foxes can lose by punishment

  • If a fox is punished (for not jumping, or not completing a jump chain), that fox is removed.
  • If there are no foxes left, foxes lose.

One Rule is not enforced

  • Chickens should not move in circles. Otherwise consider the foxes win.
